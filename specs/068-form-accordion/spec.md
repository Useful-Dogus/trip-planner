# 068-form-accordion — 편집 폼 섹션 접기

GitHub: #68

## 문제

`PanelItemForm` 이 14+ 필드 노출. 자주 수정(이름·메모) vs 드물게 수정(좌표·링크) 동등 비중. 모바일 스크롤 길어 핵심 필드 접근 비용 큼.

## 완료 조건

- "일정" / "위치" / "링크" 3 섹션을 collapsible 로 전환
- 초기 펼침 규칙: 해당 섹션 필드 중 하나라도 값 있으면 펼침, 모두 빈 값이면 접힘
- "기본 정보" + "메모" 는 항상 펼침 (헤더 클릭 불가)
- 헤더 클릭으로 토글, chevron 회전 애니메이션
- `aria-expanded` + `aria-controls`
- 접근성: 키보드(Enter/Space) 토글
- 데스크탑·모바일 동일 동작
